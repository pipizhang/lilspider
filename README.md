# Lilspider [![Build Status](https://travis-ci.org/pipizhang/lilspider.svg?branch=master)](https://travis-ci.org/pipizhang/lilspider)

### Getting started
Copy env.example to .env, put settings.

```bash
make install
source venv/bin/activate
make run
```

### Requires
* Python3
* Virtualenv
